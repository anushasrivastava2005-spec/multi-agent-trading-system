# Multi-Agent AI Trading System

AI-powered trading analysis platform built using LangChain and LangGraph.

## Features
- Multi-agent orchestration
- Technical indicator analysis
- Pattern detection
- Trend analysis
- Risk analysis
- Frontend integration

## Tech Stack
- Python
- LangChain
- LangGraph
- FastAPI
- React
- Vite
- JavaScript

## AI Agents
- Indicator Agent
- Pattern Agent
- Trend Agent
- Risk Agent

## Project Structure
- Backend services
- AI agent orchestration
- Trading analysis pipeline
- Frontend integration

## Installation

```bash
pip install -r requirements.txt
```

## Run Backend

```bash
python main.py
```

## Purpose
This project demonstrates a multi-agent AI workflow for trading analysis using modular LangGraph agents and backend orchestration.
